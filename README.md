# Full-Stack Todo Application

This project is a full-stack Todo application with a Spring Boot backend and a React frontend.

## Folder Structure
- `backend/`: Spring Boot application.
- `frontend/`: React application.

## Getting Started

### Prerequisites
- JDK 17
- Maven
- Node.js & npm

### Backend Setup
1. Navigate to `backend/`.
2. Run `mvn spring-boot:run`.
3. The API will be available at `http://localhost:8080/api/todos`.
4. H2 Console is available at `http://localhost:8080/h2-console`.

### Frontend Setup
1. Navigate to `frontend/`.
2. Run `npm install`.
3. Run `npm start`.
4. The application will be available at `http://localhost:3000`.

## Features
- Create, Read, Update, Delete (CRUD) Todos.
- In-memory H2 database.
- Modern React UI with custom hooks.
